# 코딩허브 프론트엔드 개발자 지원자 과제

본 프로젝트는 코딩허브 프론트엔드 개발자 지원자용 실무 과제입니다.

## 프로젝트 빌드 및 실행 방법

- 프로젝트를 Clone 받은 뒤 `npm start` 명령어를 실행합니다.
- [http://localhost:3000]으로 접속합니다.
